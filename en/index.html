FAQ

Answers to the most frequently asked questions about inovex elements

The inovex Elements are a collection of reusable UI components (e.g. buttons, text fields, …) for the web, based on the Web Components standard. This allows using them in any web framework, creating a consistent UI/UX while saving massive development and maintenance efforts.

Yes! The inovex Elements library is open source and available for free under the MIT license.

Web Components are a relatively new standard, supported by major browsers since 2018. Initial maintenance is more complex, but once past that hurdle you save immense effort – new components only need to be built once and work in any framework.

Install the package via npm and call defineCustomElements() in your app entry point. We also provide dedicated wrappers for Angular, React and Vue. Check our Storybook for detailed integration guides.

The inovex Elements is managed by a team of software developers and designers at inovex GmbH, a digital innovation company based in Germany.

We update the library regularly and aim to release a new version at least monthly, keeping up with the latest web technology.
