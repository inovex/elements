FAQ

Antworten auf die häufigsten Fragen rund um die inovex elements

Die inovex Elements sind eine Sammlung wiederverwendbarer UI-Komponenten (z.B. Buttons, Textfelder, …) fürs Web, basierend auf dem Web-Components-Standard. Damit lassen sie sich in jedem Framework einsetzen – für ein konsistentes UI/UX bei massiv eingespartem Entwicklungs- und Wartungsaufwand.

Ja! Die inovex Elements-Bibliothek ist Open Source und unter der MIT-Lizenz kostenlos verfügbar.

Web-Components sind ein relativ neuer Standard, seit 2018 von allen gängigen Browsern unterstützt. Die initiale Pflege ist aufwändiger, aber sobald diese Hürde genommen ist, kann immenser Wartungsaufwand gespart werden.

Das Paket per npm installieren und defineCustomElements() im App-Entry aufrufen. Für Angular, React und Vue bieten wir eigene Wrapper-Pakete. Detaillierte Anleitungen finden sich in unserem Storybook.

Die inovex Elements werden von einem Team aus Entwickler:innen und Designer:innen bei inovex GmbH, einem digitalen Innovationsunternehmen mit Sitz in Deutschland, gepflegt.

Wir aktualisieren die Bibliothek regelmäßig und streben an, mindestens monatlich eine neue Version herauszugeben.
