# Integrate with Vue.js

## Vue.js 3

Please follow these steps for integration into Vue.js(>= 3.0.0):

### Install `@inovex.de/elements-vue`

Add the package `@inovex.de/elements-vue` to your project using **npm** or **yarn**:
<CH.Code>
  ```bash npm
  npm install @inovex.de/elements-vue --save
  ```
  ```bash yarn
  yarn add @inovex.de/elements-vue
  ```
</CH.Code>

### Prepare the Project

<CH.Section>

[Import the elements module](focus://2) and [use it in your app](focus://6).

```javascript main.js
import { createApp } from 'vue';
import { InoElementsVue } from '@inovex.de/elements-vue';
import App from './App.vue'

const app = createApp(App);
app.use(InoElementsVue);
app.mount('#app');

```
</CH.Section>

### Use the Components
As the integration is now complete, you can use the components in your project in the same way you would use any other Vue.js component.
Here is an example:
<CH.Code>
```javascript app.vue
import { InoButton } from "@inovex.de/elements-vue";

export default {
    name: 'ExampleApp',
    components: {
        InoButton,
    },
    data: () => ({
        myData: [],
    }),
    methods: {
        foo() {}
    }
}
```

</CH.Code>

## Vue.js 2

Please follow these steps for integration into Vue.js(2.x.x):

### Install `@inovex.de/elements`

Add the package `@inovex.de/elements` to your project using **npm** or **yarn**:
<CH.Code>
  ```bash npm
  npm install @inovex.de/elements --save
  ```
  ```bash yarn
  yarn add @inovex.de/elements
  ```
</CH.Code>

### Prepare the Project

<CH.Section>
Before you start using the components, you need to prepare the project.

To do so, open `src/main.js` and ...
- ... [define the Custom Elements (these are the inovex elements Web Components)](focus://1:4,12:14)
- ... [tell the Vue compiler to ignore Custom Elements starting with `ino-` during compilation](focus://10)
- ... [import the icons](focus://5:8)

```javascript main.js
import {
  applyPolyfills,
  defineCustomElements,
} from '@inovex.de/elements/dist/loader';
import { addIcons } from '@inovex.de/elements/dist/collection/util/icons';
import { ICON_PATHS } from '@inovex.de/elements/dist/inovex-elements/ino-icon/index.esm.js';

addIcons(ICON_PATHS);

Vue.config.ignoredElements = [ /^ino-/ ];

applyPolyfills().then(() => {
  defineCustomElements();
});
```
</CH.Section>

### 3) Use the Components

You can now use the components in your Vue.js project in the same way you would use any native HTML element.
