# Color Palette

Our color palette is designed to support a wide range of visual cues, from primary colors to neutral shades. Utilizing CSS variables, we've created a system that makes it easy to apply consistent colors across your application.

### How to Use These Colors

To apply our color palette, reference the CSS variable names provided. For example, to use the primary colors, you can apply `color: var(--inovex-elements-primary)` or `color: var(--inovex-elements-p-6)` to your elements.
