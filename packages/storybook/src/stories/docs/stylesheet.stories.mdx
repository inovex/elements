# Stylesheet
To make your application look like the inovex-elements from top to bottom,
we have prepared a global stylesheet for you, which brings the basic styles (headings, link, theme-colors...) with it.

You can integrate it by following these simple steps:
1. Use one of our many packages (elements, elements-react, ...)
2. Import our stylsheet in your global stylesheet (e.g. like this: `@use '@inovex.de/elements/dist/inovex-elements/inovex-elements.css'`)
3. Give the body element of your application the class `inovex-elements-typo` to get access to the typography styles or the class `inovex-elements-theme` for using the elements theme colors.

With `inovex-elements-typo` the following elements now receive the inovex-elements style:
- All headlines (from `<h1>` to `<h6>`)
- [Links](/docs/docs-styleguide-typography) (`<a>`)
- The following class will be available for you to use (take a look at our typography guide for when to use what)
  - `.header-d1`
  - `.header-d2`
  - `.header-d3`
  - `.header-h1`
  - `.header-h2`
  - `.header-h3`
  - `.title-l`
  - `.title-m`
  - `.title-s`
  - `.label-l`
  - `.label-m`
  - `.label-s`
  - `.body-l`
  - `.body-m`
  - `.body-s`
